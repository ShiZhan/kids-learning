# kids-learning · 娃们的寓教于乐材料库

> 面向小学生、中学生，用游戏化、互动化的方式帮娃们巩固学习。
> 在线页面：https://shizhan.github.io/kids-learning/

## 这是什么地方？

`kids-learning` 是集中存放帮娃们学习的小工具、小游戏、练习材料的地方。目标很简单：把枯燥的练习变成能玩能练的素材，让娃们愿意多练几遍，也省家长一点力气。

后续给娃们巩固学习用的“道具”（小游戏、练习页、小测验、工具脚本等）都会统一放在这里。

## 内容分区

| 目录 | 内容 | 说明 |
|------|------|------|
| `math4kids/` | 数学练习小游戏 | 如三年级“小鱼主题”口算游戏、平移法求周长练习等 |
| `eng4kids/` | 英语练习小游戏 | 如三年级“小鱼冲关大冒险”单词/句型/拼写练习 |
| `arts4kids/` | 语文/艺术练习 | 如三年级“看拼音写词语”练习等 |

每个目录通常包含：
- `*.html`：可直接打开或部署的单页互动练习
- `*.md`：对应的设计文档、题库或思路说明
- `IFLOW.md`：该模块的迭代/执行记录（如有）

## 使用方式

1. 直接打开对应 HTML 文件即可在浏览器里玩/练；
2. 通过 GitHub Pages 自动发布到：https://shizhan.github.io/kids-learning/
3. 本地也可任意静态服务器部署，单页文件无后端依赖。

## 设计原则

- **零门槛**：单 HTML 文件为主，点开即用，无需安装。
- **轻量**：优先原生 HTML/CSS/JS，不依赖重型框架，加载快。
- **容错友好**：无失败惩罚、可反复重试，减少挫败感。
- **针对易错点**：围绕课本重点和娃们容易错的地方设计。
- **可迭代**：每个模块用 Markdown 记录需求和迭代，方便持续改进。

## 计划中的方向

- 更多年级（二、四、五、初中）的数/英/语小游戏
- 错题本、进度保存等本地复习功能
- 可打印的练习题生成工具
- 家长/老师一键查看练习情况的小工具
